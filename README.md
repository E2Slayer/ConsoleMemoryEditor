# ConsoleMemoryEditor

It is a simple console powered memory editor.


**Expected Features (Working in a Progress)**
- Enumerate all processes 32/64 bit
- Attach a Process by Process name or id
- Search Memory
- Read Memory
- Write Memory
- Make a list of memories
- Memory View
